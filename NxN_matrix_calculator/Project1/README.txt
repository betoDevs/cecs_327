Project 1

Authors: Quang Le, Alberto Avitia

Description: The purpose of this project is to multiply two square matrices of 3x3, by using a thread on each resulting element of the multiplication. In this
case we used 9 threads.

Instructions: 1. Issue comand 'make' to compile
              2. To run issue './Aprog file1 file2'. file1/file2 refers to your files where the matrices are stored.
